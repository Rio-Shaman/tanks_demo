# Самопальные танчики для Youtube
Demos for windows and linux

## Control
### Keyboard
#### Player 1
**WASD** - movement\
**Space** - shot\
**Enter** - confirm, pause\

#### Player 2
**Left, Right, Up, Down** - movement\
**Right Enter** - shot\
**Enter** - confirm, pause\

### Gamepad
**D-Pad Up/Down/Left/Right** - movement\
**Cross/A** - shot\
**Options/Start** - confirm, pause
